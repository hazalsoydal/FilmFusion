# FilmFusion
Letterboxd common watchlist movies between users🎬🍿
Our project is called FilmFusion. There is a website called Letterboxd, where you can log the movies or plan to watch. The data on this site is publicly accessible online. We created an application that shows people the movies they commonly want to watch. The project leverages Python’s Tkinter library to create an interactive and user-friendly platform. The goal of the project is to showcase our understanding of GUI development principles while providing a practical and engaging application for film enthusiasts.![image](https://github.com/user-attachments/assets/a7b23e7c-cc87-4929-9e7a-e3fc24b46ba8)
